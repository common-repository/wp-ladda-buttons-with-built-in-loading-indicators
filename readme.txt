A simple plugin to use Ladda on WordPress.

== Description ==

A set of buttons which merge loading indicators into themselves to bridge the gap between action and feedback.


Ladda buttons must be given the class "ladda-button" and the button label needs to have the "ladda-label" class.

= Accepted Data Styles =
* expand-left
* expand-right
* expand-up
* expand-down
* contract
* contract-overlay
* zoom-in
* zoom-out
* slide-left
* slide-right
* slide-up
* slide-down

= Accepted Data Colors =
* green
* red
* blue
* purple
* mint

= Accepted Data Sizes =
* xs
* s
* l
* xl

Below is an example of a button which will use the expand-right animation style.

&lt;button data-style="expand-right" data-size="l" &gt;&lt;span&gt;Submit&lt;/span&gt;&lt;/button&gt;

Demo and detail information: http://lab.hakim.se/ladda/


= Browser support =

The project is tested in Chrome and Firefox. It Should Work in the current stable releases of Chrome, Firefox, Safari as well as IE9 and up.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to Plugins > Add New > Search and search for "Ladda for WordPress").
* Alternatively, you can download the plugin from the repository.

1. Unzip and upload the files to the `/wp-content/plugins/ladda/` directory
2. Activate the "Ladda for WordPress" plugin through the 'Plugins' menu in WordPress
